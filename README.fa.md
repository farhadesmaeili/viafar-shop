# فروشگاه ویافر

🇮🇷 فارسی | 🇺🇸 [English](README.md)

---

## معرفی

ویافر شاپ یک فروشگاه اینترنتی مدرن، چندزبانه و مقیاس‌پذیر است که با **Next.js 16**، **React 19** و **TypeScript** توسعه داده می‌شود.

این پروژه با تمرکز بر معماری تمیز، قابلیت توسعه و نگهداری بلندمدت طراحی شده است.

> 🚧 پروژه در حال توسعه است.

---

## تکنولوژی‌ها

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- Base UI
- next-intl
- pnpm

---

## شروع پروژه

نصب وابستگی‌ها:

```bash
pnpm install
```

اجرای پروژه:

```bash
pnpm dev
```

ساخت نسخه Production:

```bash
pnpm build
```

بررسی کیفیت پروژه:

```bash
pnpm check
```
