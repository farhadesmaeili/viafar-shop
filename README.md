## About

Viafar Shop is a modern multilingual e-commerce platform built with **Next.js 16**, **React 19**, and **TypeScript**.

The project is designed with scalability, clean architecture, and maintainability in mind.

> 🚧 This project is currently under active development.

---

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- Base UI
- next-intl
- pnpm

---

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Build the project:

```bash
pnpm build
```

Run quality checks:

```bash
pnpm check
```
